Metemcyber: 脅威インテリジェンス流通基盤
========================================

Metemcyberとは何か？
--------------------
| Metemcyberは、良質なCTI (Cyber Threat Intelligence: 脅威情報) をブロックチェーン上で流通させるための、**脅威インテリジェンス流通基盤** です。
| ブロックチェーン上でCTIを流通させることで、可観測性が高いCTIの流通を実現を目指すとともに、これまで暗黙知であったインテリジェンス活用データを見える化を行います。


    .. raw:: html

Metemcyberでできること
----------------------
Metemcyberでは、CTIの共有 (売買) を通じて以下の特徴を実現します。

- インテリジェンスに基づく活動の影響を一か所に集約
- セキュリティ対応に関する気づきを共有
- よりActionableな脅威インテリジェンスの生産
